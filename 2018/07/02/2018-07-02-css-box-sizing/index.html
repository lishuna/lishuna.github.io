box-sizing使用

温习一下box-sizing的基本用法。

1.语法： content-box | border-box

2.说明

  • content-box 容器的实际宽度或高度=width；不包含padding和border值，是标准的盒模型。
  • border-box 容器的高度或宽度 = width + padding +border;包含padding和border值。
